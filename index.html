The purpose of this study was to test the usability of the Auckland Transport Android application. The rationale for choosing this specific application was that as a group we feel this application has the potential to be extremely helpful for any Auckland resident who uses public transport, yet as of 22nd of October 2015 it only has a 3 star out of 5 rating on the Google Play store.

The Auckland Transport Android Application on the Google Play Store lists the following features:

  • Real Time board for buses, trains, and ferries in the Auckland region.
  • Find My Stop functionality
  • Plan journeys using map-based search
  • See Live departure information for any Auckland bus-stop, train station for ferry terminal
  • Intuitive maps that show you the way to your nearest stop
  • Simple, easy to use interface

Using the above listed features as a base we created a set of tasks designed to test the described features.
